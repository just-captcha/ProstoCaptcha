🛡️ ProstoCaptcha (by JustPlugins)
ProstoCaptcha is a high-performance anti-bot solution designed specifically for Velocity proxy servers. It provides robust protection against bot attacks while maintaining a seamless experience for real players.

✨ Key Features
🧩 Interactive Quests: Verify players through in-game actions like resource mining or block interaction.

🗺️ Map-Captcha: Reliable graphical verification using maps directly in the player's hand.

🚫 Smart VPN/Proxy Detection: Built-in filtering system to block unwanted IP addresses and proxies.

🔗 Limbo Integration: Works perfectly with LimboAuth and LimboAPI for "empty" lobby setups.

🚀 Optimized for 1.16.5+: Smooth performance even under heavy bot attacks.

🎨 Full Customization: Supports gradients and complete message configuration to match your server's style.

🛠️ Installation
Drop the plugin JAR into your Velocity plugins folder.

Restart your proxy server.

Configure the settings in the generated config file (supports custom prefixes and hex colors).
